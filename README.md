# ABC Company Employee Analysis
Employee data analysis project using Python

## Project Overview
This project analyzes amployee data from ABC company, consisting of 458 rows and 9 columns. The goal is to preprocess the dataset, perform analysis and generate insights usng visualizations.

## Data Preprocessing
Corrected the "Height" column by replacing invalid values with random numbers between 150 and 180.
Handled missing valules in the salary column using median imputation
Ensured data consistency before analysis.

## Analysis Tasks
Below analyses were performed.
1. Distribution of employees across teams with percentage split.
2. Segregation of employees based on positions.
3. Identification of predominant age group.
4. Analysis of highest salary expenditure by team and position.
5. Correlation analysis between age and salary.

## Graphical Representation

Bar charts for team and position distribution.
Histogram and count plots for age analysis.
Bar charts for salary ecpenditure.
Scatter plot for correlation between age and salary.

## Key Insights

Employee distribution is relatively balanced across teams.
Majority of employees belong to 20-30 age group.
Certain positions dominate workforce distribution.
Salary expenditure is concentrated in specific teams and roles.
Weak correlation exists between age and salary.
